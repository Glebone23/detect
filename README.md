detect.js
=======================
browser & os detection

# Usage

Add detect.js to your page:

    <script src="detect.js"></script>

Retrieve the browser, version, & os:
    
    var ui = window.ui;
    console.log( ui.browser );
    console.log( ui.version );
    console.log( ui.os );
